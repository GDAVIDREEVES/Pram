You are Replit Build. Create a polished, production-quality UI for a “Profile” experience in a consumer social app for young, well-off professional (or previously professional) moms with young kids in Brooklyn. The app helps moms meet other moms to grab coffee at local cafes. The UI should feel like an app made by a large company: sophisticated, calm, curated, safety-forward, and premium (think clean typography, generous spacing, neutral palette). 

IMPORTANT
- Do NOT include scheduling/availability preferences anywhere in this Profile UI. Scheduling will be handled elsewhere in the app.
- The profile must emphasize safety, trust, and hyper-local Brooklyn neighborhood identity.
- Keep the design minimal-scroll, scannable, and warm.
- Build for mobile-first (iPhone). Use modern component patterns (cards, chips, bottom sheet modals, sticky CTA bar).
- Include light/dark mode support.

SCREENS TO BUILD (at minimum)
1) Profile View (for browsing another mom’s profile)
2) Edit Profile (for the current user)
3) A lightweight “Invite for Coffee” flow UI (bottom sheet or modal, not scheduling-based)
